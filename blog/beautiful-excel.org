表格美化原则:
**** 字体统一, 字号统一
**** 数字用千位分隔样式
**** 细项要缩进
- 不要用 A1 开始使用, 从 B2 开始
- B/C 列宽度设置为 1, 方便缩进呈现
- 最右侧空白列, 宽度设置为 3
**** 单位要自成一栏
**** 上下边框用粗线, 中间用细虚线
- 隐藏网格线 或者 整张表格背景色设置为白色
**** 文字左对齐, 数字右对齐, 慎用居中
- 列标题中, 如该列为数字列, 标题文字右对齐, 和该列数据估计一致
**** 手动输入数字用蓝色, 公式计算数字用黑色
- 如有不需要填入数据单元格, 可用 N/A (not applicable) 表示
- 重点数据行, 可填充颜色来强调
**** 凡使用的工作表, 设定名称; 凡不使用的工作表, 删之
